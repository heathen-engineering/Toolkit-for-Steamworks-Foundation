# ⚠️ THIS REPOSITORY IS DEPRECATED

**The "Toolkit for Steamworks Foundation" has been restructured into engine-specific repositories.** As part of our commitment to providing a "native" feel for every engine, we have moved away from a single monolithic foundation repo. This ensures better performance, tighter integration with engine-specific features (like Unity's ScriptableObjects or O3DE's EBus), and cleaner installation via package managers.

## 🚀 Active Foundations
Please migrate to the appropriate repository for your engine:

| Engine | Foundation Repository (FOSS) |
| :--- | :--- |
| **Unity** | [Unity Foundation for Steamworks](https://github.com/heathen-engineering/Unity-Foundation-for-Steamworks) |
| **O3DE** | [O3DE Foundation for Steamworks](https://github.com/heathen-engineering/O3DE-Foundation-for-Steamworks) |

## 🛠️ What about the Toolkit?
The **Toolkit for Steamworks** (our commercial extension with Lobbies, Matchmaking, Inventory, etc.) is being updated to sit directly on top of these new Foundations. 

* **GitHub Sponsors & Patreon:** Members continue to get access to the professional Toolkits and full source code via the private SourceRepo.
* **Compatibility:** The new Foundation architecture is designed so that upgrading to the Toolkit later requires zero changes to your existing Foundation code.
